 NAVIGATE TO JUPYTER LABS:
https://mybinder.org/v2/gh/jupyterlab/jupyterlab-demo/master?urlpath=lab/tree/demo

TO FIND THE TERMINAL IN JUPYTERLABS:
go to file / new launcher / then click on the black and white terminal icon
To run a command hit shift + enter

TO CLONE A GITHUB DIRECTORY INTO JUPYTER:
Type into the  terminal and hit shift + enter to run:
git clone https://github.com/Geo-Python-2020/exercise-1.git 
You can replace the url above with your git repo


Working with git-versioned scripts/notebooks in Jupyter Lab: https://cyberhelp.sesync.org/faq/git-and-jupyterlab.html

in Terminal, type cd <name_of_your_repository> to navigate into your repository
type git pull origin main to get current version of repo from GitHub/GitLab (See footnote 1)
open Python console or Jupyter notebook, do your work in your script or notebook file, and save it
in Terminal, type git status to check the status of your repository. You should see that your file(s) is now listed as modified under the section “Changes not staged for commit”.
stage your changes with git add <filename>
commit your changes with git commit -m "your_commit_message_here"
pull down current version of repo from GitHub/GitLab with git pull origin main
push your changes to the repo with git push origin main
